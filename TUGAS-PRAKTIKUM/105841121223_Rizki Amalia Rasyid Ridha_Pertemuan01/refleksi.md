Melalui rangkaian praktikum DevOps di semester ini, saya berharap dapat sepenuhnya mengubah cara pandang saya mengenai siklus hidup pengembangan perangkat lunak (*Software Development Life Cycle*). Selama ini, saya mungkin lebih sering berfokus pada bagaimana menulis kode yang sekadar bisa "berjalan dengan baik di laptop saya" (*works on my machine*), tanpa benar-benar memahami bagaimana kode tersebut didistribusikan, dijalankan, dan dikelola secara aman serta efisien di lingkungan *production* yang sesungguhnya. Saya berharap kelas ini memberikan wawasan yang komprehensif, tidak hanya dari aspek fundamental teoritis, tetapi juga membiasakan saya dengan budaya kolaborasi nyata di industri.

Pada akhir semester, target utama saya adalah menguasai beberapa keterampilan esensial DevOps. Pertama, saya ingin mahir menggunakan Git untuk kolaborasi tim tingkat lanjut, bukan hanya untuk *commit* sederhana. Kedua, saya ingin menguasai teknologi kontainerisasi menggunakan Docker untuk menstandardisasi *environment* pengembangan. Ketiga, dan yang paling menantang, saya sangat antusias untuk mempelajari cara merancang dan mengimplementasikan *pipeline* Continuous Integration dan Continuous Deployment (CI/CD) untuk mengotomatisasi proses *testing* dan rilis aplikasi. Keterampilan ini akan menjadi modal yang sangat berharga bagi karir saya di masa depan.
